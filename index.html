Autofill repetitive transcript request form questions

Google forms actually come with a cool feature to get a link with a pre-filled form, but it's generally restricted to form editors and not responders. You can use this method to get a pre-filled form link that helps you get through transcript request forms a bit faster. This also works with any other google form, so repetitive attendance etc could also work.

Steps:

1. Go to the transcript request google form

2. Fill in all of the non-college-specific information, remember that application type, mailing address, etc change from school to school. This just removes the need to re-type your name and such every single time

3. Go to inspect element, and then click on the console tab

4. In the console you can paste the script below to get a link with everything pre-filled

aVeryCoolVariableName = window.location.href.toString(); if (!aVeryCoolVariableName.includes('?')) { aVeryCoolVariableName += '?' } Array.from(document.querySelectorAll('[jsname="o6bZLc"]')[0].getElementsByTagName("input")).forEach((element) => { if (element.value !== '') { aVeryCoolVariableName += '&' + element.name + '=' + element.value.replaceAll(' ','+'); } }) console.log(aVeryCoolVariableName)
